## Instalasi
1. Install dependencies
```bash
composer install
npm install
```
2. build project
```bash
npm run build
```
3. buat .env dengan isi .env.example 
4. Generate key
```bash
php artisan key:generate
```
5. Jalankan migrasi
```bash
php artisan migrate
```
6. Jalankan seeder
```bash
php artisan db:seed
```
7. link ke storage
```bash
php artisan storage:link
```
8. Jalankan project
```bash
php artisan serve
```
